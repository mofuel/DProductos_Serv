Bienvenido al Manual de Usuario

Conoce cómo usar paso a paso nuestra aplicación de navegación robótica y gestión de medicamentos.

1. Inicia sesión

Ingresa tu correo y contraseña para acceder al sistema.

2. Simula la ruta del robot

Selecciona un punto de inicio y destino en el mapa. El sistema calculará la mejor ruta.

3. Gestión de medicamentos

Agrega, edita y programa medicamentos para cada paciente.

4. Programar horarios

Define a qué hora debe entregarse cada medicamento.

5. Notificaciones por correo

Recibe alertas cuando sea hora de tomar los medicamentos.

¡Y eso es todo!

Explora el sistema y mejora la experiencia de atención automatizada.
